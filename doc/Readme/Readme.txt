---------------------------------------------------------------------------------------

Bootstrap是一个免费的开源前端框架（库），用于设计网站和Web应用程序。
它包含基于HTML和CSS的版式设计模板，表单，按钮，导航和其他界面组件，以及可选
的JavaScript扩展。与许多Web框架不同，它只关注前端开发。


Bootstrap是一个用HTML，CSS和JS开发的开源工具包。使用我们的Sass变量和mixins，响应式
网格系统，大量预构建组件以及基于jQuery构建的强大插件，快速构建原型或构建整个应用程序。
